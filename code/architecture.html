Pipeline 概览

阶段目标训练模块Loss
Stage 1 跨 subject 的 contrastive pre-training，学习 subject-invariant 的 EEG 表征 cross_encoder + cross_projector InfoNCE (NT-Xent)
Stage 2 EEG embedding 对齐到 frozen text anchor，注入 task 语义 align_encoder + align_projector + logit_scale CLIPLoss (双向 CE)

设计原理

Temporal Convolution: 1D conv 沿时间轴滑动，相当于 learnable band-pass filter bank。多个 filter 捕捉不同 frequency pattern。AvgPool 做 temporal downsampling。
StratifiedLayerNorm (CLISA): 在 Temporal Block 和 Spatial Block 之间，按被试分组进行 LayerNorm。Stage 1 时将同一被试的所有 epoch 视为一组，消除 subject 间分布差异，仅保留 task-relevant 信号。Stage 2/3 时传入 n_per_subject=None，退化为 identity，不影响其他阶段。
Spatial Convolution: kernel 覆盖全部 channel，一次性做 channel mixing，将 channel 维度折叠为 1。等效于学习 spatial filter 来加权组合不同 electrode 信号。
Adaptive Pool + Projection: AdaptiveAvgPool 保证不同长度输入得到一致的 temporal dimension。Flatten 后 Linear 映射到 embed_dim。

Stage 1 Cross-Subject Contrastive Pre-training (CLISA-style)

借鉴 CLISA 的跨被试对比学习范式。对每对 train subject (A, B)，按条件和时间位置对齐采样 EEG epoch。 通过 InfoNCE 学习 subject-invariant 表征，使同条件同位置的不同被试 EEG 靠近，batch 内其他样本远离。 Encoder 内嵌 StratifiedLayerNorm，在 Stage 1 按被试分组归一化，消除个体分布差异。

采样策略: CrossSubjectPairDataset — 遍历所有 train subject pair (A, B)。对每对 subject、每个条件 (休息/心算)，找到两人共有的 epoch 位置，随机采样 segs_per_cond 个位置。A 和 B 在相同位置的 epoch 组成 positive pair；batch 内不同位置/不同条件的样本互为 negative。
StratifiedLayerNorm: 在 temporal conv 之后、spatial conv 之前，按被试分组做 LayerNorm。每个 subject 的所有 epoch 共享一组 mean/std 进行归一化，抹平个体间分布差异。仅在 Stage 1 激活 (n_per_subject≠None)，Stage 2/3 自动退化为 identity。

Stage 2 EEG ↔ Frozen Text Embedding Alignment

Text embedding 来自 frozen nn.Embedding 层: 每个 label (0=休息, 1=心算) 映射到一个固定的 proj_dim 维 anchor 向量，不训练。 EEG branch 通过 CLIPLoss 学习将 EEG representation 对齐到这些固定锚点。

Text path (frozen):
label index → nn.Embedding (frozen) → text_emb

设计说明

为什么用 frozen embedding?
每个 label 对应一个固定的 anchor 向量（随机初始化后不再更新）。EEG encoder 学习将同 label 的 EEG representation 映射到对应 anchor 附近。 这比直接训练 classifier 多了一层 contrastive 约束 — 迫使 EEG representation 在 alignment space 中形成按 label 聚簇的结构，为 Stage 3 的 fusion 提供更好的初始化。

Stage 3 Dual-Branch Classification Fine-tuning

两个 pre-trained encoder 的 embedding 拼接后通过 Fusion MLP 分类。推理时不使用 text — 纯 EEG 端到端分类。

Dual-branch fusion 结合两种互补表征:
• cross_encoder: 捕获 subject-invariant EEG pattern (通过 cross-subject contrastive 获得)
• align_encoder: 捕获 task-semantic EEG pattern (通过 text alignment 获得)
Concatenation 保留两种信号，MLP 学习加权组合用于分类。

Text Embedding 机制

当前设计: 使用 nn.Embedding(n_classes, proj_dim)，requires_grad=False。
每个 label (0=休息, 1=心算) 对应一个随机初始化后固定不变的 proj_dim 维向量，作为 alignment anchor。

为什么不训练 text embedding? 本系统目标是让 EEG representation 自身具有 task-discriminative 结构。 Frozen anchor 仅提供稳定的对齐目标。具体的向量值不重要 — 重要的是不同 label 的 anchor 互不相同，使 EEG 被迫按 label 聚簇。
